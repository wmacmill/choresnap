Adds the ability to customize the review process by adding multiple review categories and controlling the star count.
= Documentation =

Usage instructions for this plugin can be found in the Listify documentation: [http://listify.astoundify.com/category/477-reviews](http://listify.astoundify.com/category/477-reviews).

= Support Policy =

We will happily patch any confirmed bugs with this plugin, however, We will not offer support for:

1. Customisations of this plugin or any plugins it relies upon
2. CSS Styling (this is customisation work)

If you need help with customisation you will need to find and hire a developer capable of making the changes.
