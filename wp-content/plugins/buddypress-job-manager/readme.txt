== Description ==

This plugin integrates WP Job Manager and its extensions into your BuddyPress user profiles.

This plugin needs BuddyPress and WP Job Manager to be installed!

BuddyPress Job Manager is compatible with the following WP Job Manager extensions:
Applications, Bookmarks and Job Alerts.

A separate tab will be available in the user profile for every installed extension.

2. Activate the plugin through the 'Plugins' menu in WordPress
