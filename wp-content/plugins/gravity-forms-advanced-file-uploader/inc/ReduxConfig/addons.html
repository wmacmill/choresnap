Gravity Forms Adv Uploads Email Tag Addon

Gravity Forms Adv Uploads Email Tag Addon extends the Gravity Forms Advanced File Uploader plugin and provides you with a custom merge tag to add into your emails.

The custom merge tag will render a table of all files and/or external videos uploaded via each form submission.
